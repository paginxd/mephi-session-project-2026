# Проект: Администрирование рабочей станции под управлением ОС GNU/Linux

## Студент: M255621

## Цель: 
Подготовить минимальную установку Fedora 43 к работе в локальной сети, настроить безопасность и развернуть web-сервер.

## Выполненные задачи

### 1. Управление сетью

| Параметр | Значение |
|----------|---------|
| Статический IP | 192.168.1.100/24 |
| Шлюз | 192.168.1.1 |
| DNS | 8.8.8.8 |
| Имя хоста | mephi-2026.domain.local |
| Проверка связности | ping до шлюза и интернета |

> Настройка сети завершена, проверка доступности внешних ресурсов успешна.

---

### 2. Управление ПО

| Пакет / Действие | Статус |
|-----------------|--------|
| nginx | Установлен |
| tcpdump | Установлен |
| libcap-ng-utils | Установлен |
| tcpdump RPM | Локальная установка выполнена |

> Все необходимые пакеты установлены и готовы к использованию.

---

### 3. Файловая система и сервисы

| Действие | Детали |
|----------|--------|
| Раздел на /dev/sdb | Создан один раздел, занимающий всё пространство |
| Файловая система | ext4, метка MEPHI_DATA |
| Точка монтирования | /data/mephi-web, автоподключение через /etc/fstab |
| Web-сервер | nginx развернут и включен при старте системы |

> Раздел успешно примонтирован, nginx обслуживает директорию.

---

### 4. Управление доступом

#### DAC (дискреционный доступ)

| Элемент | Настройка |
|---------|-----------|
| Пользователь | mephi-admin |
| Группа | mephi-devs |
| Права на /data/mephi-web | 2775 (setgid) |

#### MAC (SELinux)

| Параметр | Значение |
|----------|---------|
| Режим | Enforcing |
| Контекст | httpd_sys_content_t для /data/mephi-web |

#### Capabilities (tcpdump)

| Настройка | Статус |
|-----------|--------|
| cap_net_raw | Добавлен |
| cap_net_admin | Добавлен |
| SUID | Убран |

> Доступы и привилегии настроены для безопасности и корректной работы сервисов.

---

### 5. Аутентификация
